Application de présentation

L'objectif de cette démo est de montrer les possibilités qu'offre le SDK a travers une IHM web hébergée dans une app. Chaque onglet présente une fonction spécifique. Dans l'ordre, on y trouve une page montrant les capacités graphiques via la librairie chart.js, puis une page démontrant l'intégration d'iframes dans l'application. L'onglet "datalayer" permet de communiquer avec le contrôleur à travers le brocker datalayer. Pour finir, une base de données SQLite a été intégrée dans l'application et vous pouvez y faire des requêtes depuis l'onglet "DataBase SQLite "

Son architecture

L'application tourne sur flask. Ce framework python permet d'héberger localement des pages web. Grâce à lui, on peu aussi communiquer avec notre base de données a travers SQLAlchemy. Si vous souhaitez installer l'application complète ou voir son code source, rendez-vous sur Github. Vous y trouverez dans le readme toute la documentation nécessaire.
